# Adaptorパターン

既存のクラスを変更することなく、新しいAPIを提供するパターン

### 種類
* 継承を使ったもの
* 委譲を使ったもの


### メリット
* アクセスクラスは処理内容を知る必要がない。
* 既に存在するクラスに変更を加える必要がない。
* 公開するAPIを制限できる。
